# xm-first-project

## 项目介绍
这是 xm 的第一个 GitHub 测试项目，用于熟悉远程仓库创建、README 规范编写、版本控制与代码托管的基础操作，练习 Git 基础命令与项目文档规范书写。

## 项目用途
本项目作为 GitHub 入门实践仓库，完成远程仓库创建、README 编写、Git 克隆/提交等基础版本控制操作，同时可用于承载后续项目的代码与文档，是个人 GitHub 开发的第一个标准化仓库。

## 使用方法
1.  复制仓库 HTTPS 地址，在本地 Git 环境执行 `git clone` 命令，将项目拉取到本地电脑；
2.  修改本地文件后，通过 `git add`、`git commit`、`git push` 指令，将更新同步到 GitHub 远程仓库。
